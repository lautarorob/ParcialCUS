Sobre HTML5

HTML5 es la última versión estándar del lenguaje de marcado para la web. Mejora la semántica, multimedia y compatibilidad con dispositivos modernos.

Curiosidades

HTML5 introdujo etiquetas como <article>, <section>, <aside> y <nav> para mejorar la estructura semántica.
